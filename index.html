How this works

  • Your logs are saved on this device (localStorage).
  • Use Export weekly so you don’t lose data if your browser clears storage.
  • If you want cloud sync later, we can connect this to Google Sheets.

GitHub Pages publish (quick steps)

  1. Create a new GitHub repository named workout-tracker.
  2. Upload these files: index.html, styles.css, app.js.
  3. In GitHub: Settings → Pages → “Deploy from a branch” → Branch: main → folder: /(root).
  4. Wait for the Pages URL, then open it on your phone.

Made for quick daily execution. If you skip a day, just resume the next scheduled workout.
